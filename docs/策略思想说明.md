## 补涨策略
- 在某个时期，相同概念中的部分个股出现了一定的涨幅，同产业链上下游的一些个股存在补涨机会；
- 比如CPO中原材料和EML芯片涨幅很高，那么可以推测的就是 产业链中 比如switch 芯片 的个股 盛科通信反而没有涨多少，那么存在补涨需求

## 产业策略
- 从产业链中不同环节存在价值量差异，优先涨价值量高的概念，后续补涨其他细分领域
- 优先涨龙头，补涨细分其他公司

## 思路细节
- agent 不可见的是思考的过程，但是agent的输出需要不断的 进行结构化呈现-比如markdown 渲染
- 我们这个系统不能一直让用户处于等待状态 需要看到 输出，包括agent 的思考过程等
- 最重要的就是 数据源（meta-data）的渲染，这是整个系统区别其他单纯的chat 工具的核心----可信推理；
