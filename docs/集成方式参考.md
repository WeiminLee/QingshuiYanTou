# 一、集成思路（最清晰）
1. **Persona 是原生 JS 组件**，不依赖任何框架
2. **TDesign 是框架组件库**，提供页面 UI
3. 集成方式：
   - 在 TDesign 页面里**放一个容器**
   - 用 JS 把 Persona 渲染到这个容器
   - 用 Persona 的**主题配置**让样式和 TDesign 完全统一
   - 用 Persona 的**插件系统**替换部分 UI 为 TDesign 组件（可选）

---

# 二、最快集成示例（以 Vue3 + TDesign 为例）
## 1. 安装 Persona
```bash
npm install @runtypelabs/persona
```

## 2. 页面代码（直接复制）
```vue
<template>
  <div>
    <!-- TDesign 正常使用 -->
    <t-page-header title="AI 助手页面" />

    <!-- Persona 容器 -->
    <div id="persona-container" style="height: 700px;"></div>
  </div>
</template>

<script setup>
import { onMounted } from 'vue'
import { createAgentExperience } from '@runtypelabs/persona'
import '@runtypelabs/persona/dist/style.css'

onMounted(() => {
  createAgentExperience('#persona-container', {
    // 👇 关键：让 Persona 样式 = TDesign 风格
    theme: {
      accent: '#0052d9',     // TDesign 主色
      success: '#00b42a',
      warning: '#ff7d00',
      danger: '#f53f3f',
      neutral: '#e5e6eb',
      background: '#ffffff',
      foreground: '#1d2129',
    },

    // 你的 AI 接口
    backend: {
      type: 'sse',
      url: '/api/ai/chat',
    },

    // 关闭默认浮动按钮，嵌入页面内
    launcher: {
      enabled: false,
    },
  })
})
</script>
```

**运行效果：**
Persona 会直接嵌入在 TDesign 页面里，**风格完全统一**，就像 TDesign 自己的组件一样。

# 三、React + TDesign 集成（同样简单）
```jsx
import { useEffect } from 'react'
import { PageHeader } from 'tdesign-react'
import { createAgentExperience } from '@runtypelabs/persona'
import '@runtypelabs/persona/dist/style.css'

export default function AiPage() {
  useEffect(() => {
    createAgentExperience('#persona-container', {
      theme: {
        accent: '#0052d9',
        success: '#00b42a',
      },
      backend: {
        type: 'sse',
        url: '/api/ai/chat',
      },
      launcher: { enabled: false },
    })
  }, [])

  return (
    <div>
      <PageHeader title="AI 助手" />
      <div id="persona-container" style={{ height: '700px' }} />
    </div>
  )
}
```

# 四、进阶：把 Persona 按钮换成 TDesign 按钮（最像原生集成）
你可以用 Persona 的**事件系统**，让 TDesign 按钮控制打开/关闭聊天窗口。

```vue
<template>
  <t-button @click="openChat">打开 AI 助手</t-button>
  <div id="persona-container"></div>
</template>

<script setup>
import { createAgentExperience } from '@runtypelabs/persona'

let persona

onMounted(() => {
  persona = createAgentExperience(..., {
    launcher: { enabled: false }
  })
})

const openChat = () => {
  persona.open() // 用 TDesign 按钮控制
}
</script>
```
